ABOUT

ゴーサインは群馬県前橋市で Web システム・アプリケーション開発や Web サイト制作を手がけている個人事務所です。

「仕事や生活をもっと楽しく便利なものに」を理念として、インターネットや IT の技術を活用して様々な課題を気持ちよく解決するお手伝いをいたします。


Web システム開発

オープンソースソフトウェアや Web サービス、Web API を活用したシステム・アプリケーション開発を手がけています。

PHP フレームワークを基盤としたスクラッチ開発、既存 CMS をカスタマイズして低コストに必要機能を実現、商用の開発ツールを利用した RAD、などご要件に適した柔軟な開発方式に対応可能です。

Web サイト制作

WordPress を利用した Web サイト制作やプラグイン開発、また外部システムとの連携等には特に力を入れており、多くの開発実績があります。

その他 Web サイトに関わるサーバ環境構築から SNS を活用した Web サイトの運用まで、様々なご要件もサポートいたします。

IT コンサルティング

IT 導入はただやれば良いというものではなく目的や方法を間違えると費用と労力を無駄に費やしてしまうリスクが潜在しています。

こんなものを作りたい、こんな困り事を解決したい・・・適切な技術やアイデアをご提供して課題を解決するお手伝いをいたします。

仕事で大事にしている考え方

  • 満足度を高める（お客様の要望に対してそれを少しでも超えるアウトプットを心がける。）
  • 自分の感覚を大事にする（思考停止してお客様や他人の意見・要望をただこなすだけの作業はしない。）
  • できるだけ作らない（使えるものは活用して本当に必要な事にリソースを割く。作ることが目的にならないように。）
  • 楽しい方を選択する（迷った時は面白そうと感じられる選択肢を選ぶ。楽しめる作業は良い成果にも繋がる。）
  • 頑張る時は頑張る（ハードワークの先に見える世界もある。楽できるに越したことはないがやるときはやる。）
